# Garner Quant V2.1

Garner Quant is a Python-based multi-asset investment research and paper trading system.

It analyses:
- Global ETFs
- Individual equities
- Gold
- Crypto assets

It uses:
- EMA
- RSI
- MACD
- ATR
- Fundamental scoring
- ATR-based position sizing
- Paper trading
- Trade analytics
- Telegram alerts
- Broker account simulation

## Run manually

```bash
python main_v2.py
